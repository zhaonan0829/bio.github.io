Vizor 一叶故事荟 .pdf

在与影视、教育、出版等行业的多年合作中，我们逐步发现了一些它们在针对文本高效研读及客观理解方面的阻碍。于是，我们设计了一个工具，以此来提升和培养文本研究者的研读效率及客观理解文本的能力。

我们通过分析反馈、咨询专家、读取用户日志的方式为产品指明方向，并在初始版发布之后就开始着手设计系统建设，以此来协调与规范前端、交互、视觉的行为。

为了使用户的学习曲线不过于陡峭，同时确保新功能能够增加用户体验而非减少它。这也就是为什么我们对添加特性保持克制的态度，以及为什么对构建特性如此关注的原因。

短视频 .pdf

区别于市面上其他常见的短视频 APP，这款融合 PGC 与 UCG 两种模式的短视频应用专为海外用户定制设计。

我们从「背景研究」和「竞品分析」开始，了解了行业概况和平台现状，到通过建立「利益相关者地图」明确用户汇报关系，并帮助我们在不同阶段引入不同的角色。在收集「商业需求」与「用户需求」后，我们开始梳理用户研究结果，从各个维度梳理现有业务，提出问题，展开一系列的头脑风暴，形成优化后的涵盖各关键业务流程的新体验地图。

在这之后，我们开始关键原型及动效设计呈现，并开始用户测试。最终撰写涵盖各参与角色的开发故事，为之后的项目排期和技术落地做准备。

TFlow 印前自动化（PROCESS）.pdf

团队凭借十余年的行业经验，通过深入印刷企业基层的实地观察，结合与印前人员的互动，我们发现并印证了一些长期困扰印企的实际难题。

经过系统的调研后，我们决定为他们设计一套易用、易维护的印前流程化管理系统， 它可以完成从商城接单（云印平台）– 工单拾取、预飞 – 自动拼板（Phoenix）– 在线编辑的服务生态循环。 为了获得那些在业界广受赞誉的团队支持，我们拜访了在比利时的 Chili 与 Enfocus switch 工作室，他们最终成为了我们的伙伴。

中国移动 - 物联网企业链接管理云平台（PROCESS） .pdf

移动物联作为新一代信息技术的重要组成部分，在日益激增的用户及需求驱动下，中国移动仅为企业提供 接口对接式 管理已经不足以满足用户需求。

在这个项目中，我们与移物联网专家一同深入洞察了依赖于中国移动物联的几个重要企业用户（OFO单车、比亚迪等）的物联卡管理流程，基于这些流程的共性，我们为广东省移动设计了一个新的网站，以 WEB 门户和 API 接口两种方式，面向物联网业务企业客户提供 大规模、自助化、全生命周期的通信连接管理服务。

科文微讲堂（概念） UPCOMING

精神文明建设说到底是做凝聚人、引领人、培育人的工作，最大特点在于群众性，而全民阅读正是推动精神文明建设群众性的一个方面。国家重视文化产业的发展，有声读物也获得政府的大力扶持，尤其是基于移动互联网的有声书平台正迎来新的发展机遇。有声读物作为文化产品，在推动精神文明建设和物质文明建设以及社会进步方面产生了巨大的效用，而这一效用却常常为人忽视。

如今，得到上海科技文献出版社的鼎力支持，我们意图建立起以全社版权书为基础的科普知识音频数据库，力求帮助盲人、青少年乃至全民接受广泛的科普教育。
